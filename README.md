<h1 align=center> FFmpeg GUI Tool </h1>

### *Work in progress*

A simple GUI tool built on PyQt6 for viewing audio and video metadata and performing actions without needing to use the command line.

### Features:
- Load individual files
- View stream and format data

### Planned Features:
- Load and process multiple files
- Convert encoding
- Trim
- Change metadata
- Delay
- Change volume
- Re-encode (video only)
- Burn subtitles (video only)
- Remove some/all subtitles (video only)
- Burn frame number (video only)
- Concatenate (audio only)
